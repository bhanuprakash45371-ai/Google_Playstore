# Google Play Store Analysis

## Project Overview

This project focuses on analyzing Google Play Store applications and user reviews to understand app market dynamics, user preferences, app performance, and customer sentiment. The analysis leverages data cleaning, exploratory data analysis (EDA), sentiment analysis, and data visualization techniques to uncover meaningful insights from the Android app ecosystem.

---

## Objective

The primary objective of this project is to:

* Clean and preprocess Google Play Store data.
* Analyze app categories and market distribution.
* Study app ratings, reviews, installs, and pricing trends.
* Perform sentiment analysis on user reviews.
* Generate meaningful visualizations and business insights.
* Support data-driven decision-making for app developers and businesses.

---

## Dataset Information

### Dataset 1: Apps Data

Contains information about applications available on the Google Play Store.

Features include:

* App Name
* Category
* Rating
* Reviews
* Size
* Installs
* Type (Free/Paid)
* Price
* Content Rating
* Genres
* Android Version
* Current Version

### Dataset 2: User Reviews

Contains user-generated reviews and sentiment information.

Features include:

* App Name
* Review Text
* Sentiment
* Sentiment Polarity
* Sentiment Subjectivity

## Key Insights

### App Categories

* Family and Game categories dominate the Play Store ecosystem.
* Tools and Productivity apps maintain strong market presence.

### Ratings Analysis

* Most applications have ratings between 4.0 and 4.5.
* User satisfaction is generally positive across categories.

### Free vs Paid Apps

* Free applications significantly outnumber paid applications.
* Freemium business models remain highly popular.

### Reviews and Installs

* Reviews and installs show a moderate positive correlation.
* Apps with higher installs tend to receive more user reviews.

### Sentiment Analysis

* Positive reviews account for the majority of user feedback.
* Negative reviews highlight opportunities for improvement.
* Sentiment analysis provides deeper insights than ratings alone.

## Recommendations

1. Focus on improving user experience and app stability.
2. Regularly monitor negative reviews to identify issues.
3. Encourage user engagement through updates and new features.
4. Optimize app performance to improve ratings and retention.
5. Combine sentiment analysis with rating analysis for better decision-making.
